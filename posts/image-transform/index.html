图像平移矩阵

September 23, 2019
计算机视觉

图像乘以单应性矩阵H进行透视变换时 H为单位矩阵，变换后的图和原图一致 H[0][3]的值代表右移尺度 H[1][3]的值代表下移尺度

右移20像素

img1 = cv2.imread("a.jpg", cv2.IMREAD_COLOR)
h, w, _ = img1.shape

H = np.array([[1.0, 0, 20],
              [0, 1.0, 0],
              [0, 0, 1.0]])

wrap1 = cv2.warpPerspective(img1, H, (w, h))
cv2.imwrite("a_.jpg", wrap1)

下移20像素

img1 = cv2.imread("a.jpg", cv2.IMREAD_COLOR)
h, w, _ = img1.shape

H = np.array([[1.0, 0, 0],
              [0, 1.0, 20],
              [0, 0, 1.0]])

wrap1 = cv2.warpPerspective(img1, H, (w, h))
cv2.imwrite("a_.jpg", wrap1)

对应值为负分别代表左移和上移
